# turtlepose
## 功能說明
透過鍵盤控制烏龜移動，並在介面上顯示目前烏龜的位置狀態，每200ms更新一次位置資訊。

##執行
 1. 確認[編譯](https://github.com/EdXian/ROS_QT_GUI/blob/master/README.md)完成。
 2. `roslaunch turtlepose tuetlepose.launch`
 3. 程式請在終端機輸入`Ctrl + c`
